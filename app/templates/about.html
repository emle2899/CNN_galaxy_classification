About the Project

This is a student project by Emily Levis for the Galvanize Data Science Immersive Program

Abstract

Galaxies are one of the most visually stunning cosmic phenomenon and with Sloan Digital Sky Survey and Hubble, we now have thousands of photometry of galaxies to admire. The problem for astronomers, however, is classifying these galaxy images. One solution is using neural networks to classify these images for us. This project uses a Convolutional Neural Network (CNN) to classify galaxies, which is a tool already used in many galaxy classifications. This is made possible by projects like Galaxy Zoo/Zooniverse, which asked volunteers to label thousands of galaxies through a series of questions. Thanks to this, there are archives of ton of labeled data to train on.

The data originally comes from galaxy zoo - a project where volunteers labeled thousands of galaxies through a series of eleven questions. This version of the data is in jpg files from Kaggle. For this project, the computer is identifying spirals and ellipticals, with spirals split into spiral, barred-spirals, and edge-view spirals. While edge-view spirals are not cosmologically different, they are visually different for both the viewer and the computer, hence the separate category. I set up the labels by categorizing user responses by majority percentage vote. As such, there is potential human error in the actual labels.
If you would like to know more about the project, please visit my Github page.
